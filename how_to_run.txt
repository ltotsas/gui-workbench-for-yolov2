In order to run the software both detector-api and detector-app must be build.
On each folder inside execute "npm install", npm build and npm run start
The interface will be available at "http://localhost:4200"

For using yolo the current implementation is using conda as virtual environment, thus it is a must.
Therefore, anaconda installation must be present.
On the root of the folder where the "environment.yml" file is, execute "conda env create -f environment.yml"

Lastly, the weights of the backend Yolo must be downloaded and placed inside the folder Yolov2
https://drive.google.com/file/d/1Vk4wXj4EfNOZxvQJNR5WhKgiZlhqrhRy/view?usp=sharing
